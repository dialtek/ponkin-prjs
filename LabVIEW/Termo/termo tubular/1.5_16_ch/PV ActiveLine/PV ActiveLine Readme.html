Pfeiffer Vacuum Instrument Driver Readme

To rate this instrument driver, go to the driver download page on the Pfeiffer Vacuum homepage.

1. Overview

Instrument Driver Technology: LabVIEW Plug and Play (project-style)
Manufacturer: Pfeiffer Vacuum GmbH
Supported Language(s): LabView 2013SR1
Supported Model(s): Pfeiffer Vacuum DualGauge (TPG 252), (TPG262), (TPG362) and SingleGauge (TPG261), (TPG361)

Model(s) Tested: DualGauge (TPG 252), (TPG262), (TPG362) and SingleGauge (TPG261), (TPG361)
Interface(s): Serial (RS232), USB.
Firmware Revision(s) Tested: DualGauge (TPG 362), Single Gauge (TPG 361)

Certified: No
NI Supported: No
Source Code Available: Yes

Driver Revision: 2015
Original Release Date: 10.02.2013
Current Revision Date: 10.05.2013

2. Required Software

Some software components need to be installed before using this instrument driver. The minimum versions of these components are listed below, and can be downloaded from the Download Site.

VISA 4.4 or later

LabView 2013 or later

Refer to the LabVIEW Help for more information about software requirements. You access the LabVIEW Help by selecting Help»Search the LabVIEW Help.
